Projects

Vrundavan Dairy Farm

The Vrundavan Dairy Farm Management System is a web-based application designed to streamline the management of dairy products. It enables users to perform CRUD (Create, Read, Update, Delete) operations on products, including uploading product images. The frontend is developed using React.js, while the backend is powered by Node.js and Express.js, with MySQL as the database. The system ensures secure and efficient data handling, utilizing Multer for image uploads and Joi for data validation. This project follows a RESTful API architecture to maintain scalability and ease of integration.
Technologies : Node.js, React.js, MySQL

Nakya Resarch

Nakya is a web-based platform focused on secure API management. It includes authentication, authorization, data validation, and MySQL integration. APIs were developed using Node.js and Express.js, with Joi for validation. Testing was done via Postman to ensure reliability.
Technologies : Node.js, MySQL
